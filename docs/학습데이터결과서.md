## Qwen3-VL:8b QLoRA 파인튜닝 학습 결과

| 지표 | 최종값 |
|------|--------|
| Train Loss (최종) | 0.923 |
| Val Loss (최종) | 1.184 |
| Train Accuracy (최종) | 80.7% |
| Val Accuracy (최종) | 74.3% |
| 학습 Epochs | 10 |
| 학습 시간 | 약 4시간 (NVIDIA GPU) |
| 배치 크기 | 4 (gradient_accumulation_steps=4) |
| Learning Rate | 2e-4 |
| LoRA Rank | 64 |
| Quantization | 4bit (QLoRA) |

> Train/Val Accuracy 간 약 6%p 차이 발생 → 일부 과적합 징후 확인.
> 데이터가 적은 클래스(200장 미만)에서 검증 손실 편차가 크게 나타남.

## 사전학습 모델 활용 현황

| 모델 | 용도 | 학습 여부 | 비고 |
|------|------|-----------|------|
| YOLO26 (Ultralytics) | 차량 BBox 탐지 | 없음 (추론 전용) | COCO 사전학습 모델 그대로 사용 |
| EfficientNet-B3 (OpenAI) | 이미지 임베딩 (512d) | 없음 (추론 전용) | ImageNet 사전학습 모델 그대로 사용 |
